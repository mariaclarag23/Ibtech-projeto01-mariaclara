# Cartão de Visita Pessoal — Maria Clara Guimarães

Projeto desenvolvido para a Trilha Frontend da IbTech 2026.1.

## Descrição

Este projeto consiste em uma página pessoal em formato de cartão de visita digital, desenvolvida com HTML, CSS e JavaScript puros.

A página apresenta informações sobre mim, minhas habilidades, links profissionais, galeria de imagens e interações dinâmicas como dark mode, terminal interativo e animações de entrada.

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript Vanilla
- Flexbox
- CSS Grid
- LocalStorage
- IntersectionObserver
- Clipboard API

## Funcionalidades

- Tema claro e escuro
- Botão para copiar e-mail
- Efeito typewriter
- Terminal interativo
- Layout responsivo
- Animações ao rolar a página
- Galeria de imagens
